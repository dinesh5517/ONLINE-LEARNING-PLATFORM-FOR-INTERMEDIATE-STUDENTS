Matrices - EAMCET Short Notes

1. Definition of Matrices

  • A matrix is a rectangular array of numbers arranged in rows and columns.
  • Notation: A matrix is usually denoted by uppercase letters (e.g., A, B, C).
  • Order of Matrix: The order of a matrix is defined by the number of rows and columns it has, denoted as m x n.

2. Types of Matrices

  • Row Matrix: A matrix with a single row (1 x n).
  • Column Matrix: A matrix with a single column (m x 1).
  • Square Matrix: A matrix with the same number of rows and columns (n x n).
  • Zero Matrix: A matrix in which all elements are zero.
  • Identity Matrix: A square matrix with 1s on the diagonal and 0s elsewhere.

3. Matrix Operations

  • Addition: Two matrices can be added if they have the same order.
  • Subtraction: Similar to addition, matrices must have the same order.
  • Multiplication: A matrix of order m x n can be multiplied by a matrix of order n x p, resulting in a matrix of order m x p.
If A = [aij] and B = [bij], then A + B = [aij + bij]

4. Determinants

  • Definition: A scalar value that can be computed from the elements of a square matrix.
  • Properties:
    • If two rows (or columns) of a determinant are equal, its value is zero.
    • The determinant of a product of matrices is the product of their determinants.
For a 2x2 matrix A = [[a, b], [c, d]], Det(A) = ad - bc
