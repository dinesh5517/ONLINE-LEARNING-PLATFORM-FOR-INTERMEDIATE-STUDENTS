1. Position, Distance & Displacement

Position: Location of a body with respect to an origin, measured along coordinate axes.

Distance: Total path length covered by a body. It is a scalar quantity.

Displacement: Shortest distance between initial and final positions with direction. It is a vector quantity.

Displacement (s) = Final position - Initial position

2. Speed & Velocity

Speed: Rate of change of distance with respect to time. It is a scalar quantity.

Velocity: Rate of change of displacement with respect to time. It is a vector quantity.

Average Speed = Total distance / Total time
Average Velocity = Total displacement / Total time
Instantaneous Velocity (v) = ds/dt

3. Acceleration

Acceleration: Rate of change of velocity with respect to time. It is a vector quantity.

Average Acceleration = (Final velocity - Initial velocity) / Time taken
aavg = (vf - vi) / (tf - ti)
Instantaneous Acceleration (a) = dv/dt
