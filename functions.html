1. Definition and Basics

  • A function f: A ‚Üí B is a relation that assigns each element of set A to exactly one element of set B.
  • Domain: Set of all possible input values (A)
  • Codomain: Set of possible output values (B)
  • Range: Actual set of output values produced by the function
  • Types of functions based on elements:
    • One-one (Injective): Each element in B has at most one pre-image in A
    • Onto (Surjective): Range = Codomain
    • Bijective: Both one-one and onto
