Applications of Derivatives - EAMCET Short Notes

1. Rate of Change

  • Derivatives measure instantaneous rate of change
  • Physical interpretation: Velocity is derivative of position
  • Economics: Marginal cost is derivative of total cost
Example:
If s(t) = t¬≥ - 4t¬≤ + 5t (position function)

3. Increasing/Decreasing Functions

Condition Interpretation
f'(x) > 0 Function is increasing
f'(x) < 0 Function is decreasing
f'(x) = 0 Critical point (possible extrema)
Visualization:

Increasing functions rise to the right, decreasing functions fall

4. Maxima and Minima

First Derivative Test:
  1. Find critical points (f'(x)=0 or undefined)
  2. Analyze sign change of f'(x):
    • (+) to (-) ‚áí local maximum
    • (-) to (+) ‚áí local minimum
Second Derivative Test:
  1. Find critical points where f'(x)=0
  2. Evaluate f''(x):
    • f''(x) > 0 ‚áí local minimum
    • f''(x) < 0 ‚áí local maximum

5. Optimization Problems

  1. Identify quantity to maximize/minimize
  2. Write equation relating variables
  3. Express quantity as function of one variable
  4. Find critical points and verify maxima/minima
Example (Rectangle):
Maximize area A = xy with perimeter P = 2x + 2y = 100
y = 50 - x ‚áí A = x(50 - x) = 50x - x¬≤
dA/dx = 50 - 2x = 0 ‚áí x = 25
Maximum area when x = y = 25 (square)

6. Concavity and Points of Inflection

Condition Interpretation
f''(x) > 0 Concave upwards (holds water)
f''(x) < 0 Concave downwards (spills water)
f''(x) = 0 Possible point of inflection

7. Related Rates

  1. Identify all given rates and desired rate
  2. Find equation relating variables
  3. Differentiate implicitly with respect to time (t)
  4. Substitute known values and solve
Example (Ladder):
A 5m ladder slides away from wall at 0.5m/s. How fast is top falling when base is 3m from wall?
x¬≤ + y¬≤ = 25 ‚áí 2x(dx/dt) + 2y(dy/dt) = 0
When x=3, y=4, dx/dt=0.5 ‚áí dy/dt = -0.375 m/s

Tips for EAMCET

  • Always draw diagrams for optimization/related rates problems
  • Verify critical points are maxima/minima using tests
  • Remember units in applied problems
  • Practice setting up equations from word problems
  • Check endpoints for absolute maxima/minima
  • Master both derivative tests for maxima/minima
